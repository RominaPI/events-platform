# events-platform
Events Platform

Este proyecto es una aplicación web creada con React + Vite que muestra una lista de eventos (obtenidos desde una API externa) y sirve como práctica para aprender los conceptos básicos de React.

Tecnologías

React
-Vite
-JavaScript (JSX)
-HTML & CSS
-Fetch API

Instalación:

Clona el repositorio:

git clone https://github.com/RominaPI/events-platform.git
cd events-platform/web

Instala dependencias:

    npm install

Corre el proyecto:

    npm run dev

Abre en tu navegador:

http://localhost:5173

Estructura del proyecto:
events-platform/
 └─ web/
    └─ src/
       ├─ components/
       │   ├─ Header.jsx
       │   └─ EventList.jsx
       └─ App.jsx

Funcionalidades:

-Muestra un encabezado dinámico usando props.
-Obtiene datos desde una API externa usando fetch.
-Usa useState para manejar estado local.
-Usa useEffect para ejecutar código al cargar el componente.
-Maneja estados de carga y error.

Aprendizajes:

-Qué es un componente en React.
-Uso de props para pasar información entre componentes.
-Manejo de estado con useState.
-Uso de efectos con useEffect.
-Consumo de APIs desde React.
-Organización de componentes.
